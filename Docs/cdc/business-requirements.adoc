=== üéØ Project Overview

**BlaBlaBook V2** is a modern web platform designed for all reading enthusiasts. It allows users to create and manage their personal library, track their readings, discover new books, and share their reviews with other readers.

**Goals:**
* Foster book discussions and exchanges
* Create an active reading community
* Make reading management simple and enjoyable

**V1 ‚Üí V2 Migration:**
This V2 represents a complete rewrite with a modern technical stack to address V1 architecture limitations.

=== üåü Features by Version

==== MVP (Minimum Viable Product) - V2.0

**Core features:**

* **Homepage** : BlaBlaBook presentation + random books for discovery
* **Secure authentication** : Registration/login with stateless JWT
* **Personal library** :
  - Add books to personal collection
  - Organize by reading status (read, to-read, currently reading)
  - Remove books without global deletion
* **Optimized search** : Search engine using Open Library API
* **Book details** : Complete page with information, summary, metadata

==== Version 2.1 - Quick Improvements

**Enhanced user experience:**

* **Rating/Review system** : 1-5 star ratings + text comments
* **Personal statistics** :
  - Books read per month/year
  - Favorite genres with charts
  - Reading progress tracking
* **Advanced search** : Filters by genre, average rating, publication date
* **Dynamic search engine** : Real-time suggestions, autocompletion

**Compliance and administration:**

* **GDPR data export** : Complete user data download
* **Right to be forgotten** : Account + associated data deletion
* **Legal notices** : Complete privacy policy
* **Admin moderation** : Tools for inappropriate content management

* **Needs** : Easily track readings, discover popular books
